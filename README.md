# tslox

Lox 是作者 [Bob Nystrom](https://github.com/munificent) 所著 [《Crafting Interpreters》](https://craftinginterpreters.com/) 一书中描述的一种小型脚本语言。作为学习编译原理的实践练习，我根据书中内容以 Typescript 实现 Lox 的树形解释器。

### 快速开始

1. 安装依赖

```
pnpm install
```

2. 运行解释器

```
pnpm run lox
```

3. 执行脚本

```
pnpm run lox script/test.lox
```
